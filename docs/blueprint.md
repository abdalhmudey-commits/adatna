# **App Name**: Habitual

## Core Features:

- Habit Definition: Allow users to define the habit they want to quit, including a description and a personalized message.
- Scheduled Notifications: Enable users to schedule recurring notifications with customizable intervals (seconds, minutes, hours, or days).
- Custom Reminders: Allow users to record custom voice alerts as an alternative to on-screen reminders.
- Habit List: Display all added habits in an organized list, showing the habit description and reminder frequency.
- Habit Deletion: Allow users to delete habits, thereby stopping the notifications. A confirmation dialog is recommended.

## Style Guidelines:

- Primary color: Soft Blue (#89B2E8). Represents tranquility and focus, aiding users in staying calm and committed to breaking habits.
- Background color: Light Gray (#F0F4F8). Offers a clean and unobtrusive backdrop, ensuring readability and reducing visual strain.
- Accent color: Pale Yellow (#E8D989). Provides gentle highlights, drawing attention to key interactive elements without being overwhelming.
- Body and headline font: 'Inter' sans-serif for a modern, machined look, ideal for both headlines and body text.
- Simple, clean icons that visually represent different habits and notification settings, ensuring clarity and ease of use.
- Clean and straightforward design. Utilizing ShadCN components for a polished and accessible user interface. Primary language support: Arabic.
